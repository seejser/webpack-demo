# 前端快速开发测试脚手架

## 支持功能

- 已集成 jQuery ,全部使用，并分离单独打包

- 支持多页面 在html添加

- 支持css图片和html图片

- 支持css,sass,less,支持文件分离

- 支持开发环境和生产环境

- 支持打包后预览

- 支持开发环境热更新（浏览器字段刷新）

- 支持 es6转es5

- 使用source-map追踪源代码错误

- source-map追踪源代码错误

- 可以使用ejs实现模板化（待实现）

##  开发

```
 yarn dev

```

##  打包

```
 yarn build

```

##  预览

```
 yarn start

```
